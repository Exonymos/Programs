Example 5

An absolute file reference specifies the full URL or file path to the file being linked to, starting from the root of the website. For example:
<img src="https://example.com/images/image.jpg" alt="Example Image">

A relative file reference, on the other hand, specifies the file path relative to the current file location. For example:
<img src="../images/image.jpg" alt="Example Image">

Example 7

Get Requests:

  • Can be cached
  • Remain in browser history
  • Should only be used to retrieve data
  • Have length Restrictions
  • Should not be used for sensitive data
  • Can be bookmarked

Post Method:

  • Requests are never cached
  • Do not remain in browser history
  • Have no restrictions on data length
  • Cannot be bookmarked
